Pour faire le calcul de la valeur d'un neuronne, cela demandera que tu sache faire un produit matriciel.

question: >

Quel est le résultat de : (0, 1, 0, 1, 1) * ((1, 2, 3, 4, 5), (-1, 2, 4, 5, 3), (51, 2, -42, 4, 5))

answer: >
help : > Si tu as oublié le calcul, jette un oeil à la page wikipedia.
Les détail de calcul pour la première valeur est par exemple : 0 * 1 + 1 * 2 + 0 * 3 + 1 * 4 + 1 * 5 = 11 correction: > Le résultat est (11, 10, 11). Détail des calculs :
(0 * 1 + 1 * 2 + 0 * 3 + 1 * 4 + 1 * 5,
0 * (-1) + 1 * 2 + 0 * 4 + 1 * 5 + 1 * 3,
0 * 51 + 1 * 2 + 0 * (-42) + 1 * 4 + 1 * 5) = (11, 10, 11) ---
